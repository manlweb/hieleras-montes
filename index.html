Pagos

SPEI o link de Mercado Pago

Para seguridad del cliente y del negocio, los datos de pago se envían por WhatsApp después de confirmar modelo, disponibilidad y ciudad de envío.

Regla principal

No se realiza ningún envío sin pago confirmado del producto. Las capturas o comprobantes se revisan, pero el pedido se prepara hasta que el pago aparezca acreditado.

Pago por SPEI

La transferencia se realiza a la cuenta indicada por WhatsApp después de confirmar el pedido.

Link de Mercado Pago

Hieleras Montes puede enviar un link de Mercado Pago para que el cliente pague en línea.

Sin pago contra entrega

ODM se usa como mensajería. El producto se paga directo a Hieleras Montes antes del envío.

Mensajería

Envíos por ODM ocurre

La hielera se manda a oficina ODM para que el cliente la recoja. El costo de envío se confirma antes de cerrar el pedido.

Importante sobre el envío

La cotización de las hieleras es más envío. La cobertura, oficina de entrega, tiempo estimado y costo final dependen de ODM.

Preguntas frecuentes

¿La cotización incluye envío?

No. La cotización del producto se maneja más envío, salvo que se acuerde otra modalidad por WhatsApp.

¿Hacen envíos a toda la República?

Se realizan envíos por ODM ocurre. La cobertura y costo se confirman antes de cerrar el pedido.

¿Puedo pagar al recibir?

No. El producto debe estar pagado y confirmado antes de enviarse.

¿Qué formas de pago aceptan?

Pago por SPEI y link de Mercado Pago, después de confirmar disponibilidad.

¿Me mandan guía?

Sí. Una vez realizado el envío, se comparte guía o comprobante correspondiente.
